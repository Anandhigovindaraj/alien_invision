## 🎮 Features
- Player-controlled spaceship that fires bullets  
- Fleet of alien enemies that move and descend  
- Score tracking  
- Player lives system  
- Basic collision detection  
- Clean and simple graphics

## 🛠️ Technologies Used
- Python
- Pygame
